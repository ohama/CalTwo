<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
Complete calculator UI with CSS Grid layout, hover/active button feedback, keyboard support, and responsive design. Specifically:
- 4-column button grid (C, BS, _, ÷ / 7,8,9,× / 4,5,6,- / 1,2,3,+ / 0[span 2],.,=)
- Orange operator buttons, green equals, red clear, gray backspace
- Hover: highlight + scale up (1.05x). Active: darken + scale down (0.95x)
- Keyboard: digits 0-9, +, -, *, /, Enter (=), Escape (C), Backspace, . (decimal)
- Responsive: centered at max 400px, fits 375px mobile width with 44px+ touch targets
  </what-built>
  <how-to-verify>
1. Run `npm run dev` if not already running
2. Open http://localhost:5173 in browser

**Layout check:**
- Confirm 4-column grid layout with buttons properly aligned
- Confirm 0 button spans 2 columns in bottom row
- Confirm display area shows "0" with dark background and green text

**Visual feedback check:**
- Hover over any number button — should highlight and scale up slightly
- Click any button — should darken and scale down briefly
- Hover over orange operator button — should darken to deeper orange
- Hover over green equals — should darken to deeper green
- Hover over red clear — should darken to deeper red

**Keyboard check:**
- Click on the calculator area first (to focus it)
- Press digit keys (0-9) — numbers appear on display
- Press + then another number then Enter — correct result shown
- Press Escape — calculator clears to "0"
- Press Backspace — last character removed
- Press / — division operator activated (browser search bar should NOT open)

**Responsive check:**
- Open browser DevTools (F12)
- Toggle device toolbar (Ctrl+Shift+M or Cmd+Shift+M)
- Set viewport to 375px width (iPhone SE)
- Confirm all buttons are visible and tappable (no horizontal scroll)
- Confirm buttons are at least 44px tall
- Set viewport to 1920px — confirm calculator is centered, max 400px wide

**Functional check:**
- Type: 2 + 3 = (using buttons) — display shows 5
- Type: 1 0 ÷ 0 = — display shows Error
- Press C — display resets to 0
- Type: 1 2 3 then Backspace — display shows 12
  </how-to-verify>
  <resume-signal>Type "approved" if all checks pass, or describe issues found</resume-signal>
</task>
